## 📌 Completed Tasks

### ✅ Task 1 - Spam SMS Detection

Developed a Machine Learning model to classify SMS messages as **Spam** or **Normal Message** using **Multinomial Naive Bayes** and **CountVectorizer**.

**Accuracy:** 98.26%

📁 Folder:
`Task_4_Spam_SMS_Detection`

---

### ✅ Task 2 - Credit Card Fraud Detection

Built a Machine Learning model to classify credit card transactions as **Legitimate** or **Fraudulent** using:

- Logistic Regression
- Decision Tree
- Random Forest

The project also includes a manual prediction interface where users can enter transaction details and predict whether the transaction is legitimate or fraudulent.

📁 Folder:
`Task_2_Credit_Card_Fraud_Detection`

---

### ✅ Task 3 - Customer Churn Prediction

Developed a Machine Learning model to predict whether a bank customer is likely to **churn (leave the bank)** or **stay**.

The project uses customer demographic and banking details to train a classification model and provides a manual prediction interface where users can enter customer details and get real-time churn predictions.

**Model Used:**
- Random Forest Classifier

**Features Used:**
- Credit Score
- Geography
- Gender
- Age
- Tenure
- Balance
- Number of Products
- Credit Card Status
- Active Membership
- Estimated Salary

📁 Folder:
`Task_3_Customer_Churn_Prediction`
