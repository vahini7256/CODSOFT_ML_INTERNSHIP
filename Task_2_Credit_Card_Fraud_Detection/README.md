# 💳 Credit Card Fraud Detection using Machine Learning

## 📖 Project Overview

This project was developed as part of the **CodSoft Machine Learning Internship**.

The objective of this project is to detect whether a credit card transaction is **Legitimate** or **Fraudulent** using Machine Learning algorithms.

---

## 🎯 Objective

To build a Machine Learning model capable of classifying credit card transactions as fraudulent or legitimate using transaction-related information.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Imbalanced-learn (SMOTE)
- Jupyter Notebook (VS Code)

---

## 📂 Dataset

The project uses the Credit Card Fraud Detection dataset provided for the internship.

Files used:

- fraudTrain.csv
- fraudTest.csv

---

## ⚙️ Machine Learning Workflow

1. Import Libraries
2. Load Dataset
3. Data Preprocessing
4. Feature Selection
5. Handle Class Imbalance using SMOTE
6. Train Logistic Regression Model
7. Train Decision Tree Model
8. Train Random Forest Model
9. Evaluate Model Performance
10. Predict Custom Transactions

---

## 🤖 Algorithms Used

- Logistic Regression
- Decision Tree
- Random Forest Classifier

---

## 💻 Manual Transaction Prediction

The notebook includes an interactive prediction interface where users can manually enter:

- Transaction Amount
- City Population
- Customer Latitude
- Customer Longitude
- Merchant Latitude
- Merchant Longitude
- Unix Time

The model predicts whether the transaction is:

- ✅ Legitimate Transaction
- 🚨 Fraudulent Transaction
