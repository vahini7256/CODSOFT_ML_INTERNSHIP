# Dataset Information

## Credit Card Fraud Detection Dataset

This project uses the **Credit Card Fraud Detection Dataset** to build a machine learning model for identifying fraudulent transactions.

## Dataset Files

The following dataset files are used in this project:

- `fraudTrain.csv` - Training dataset used for model training.
- `fraudTest.csv` - Testing dataset used for model evaluation.

## Dataset Source

The dataset is obtained from publicly available machine learning resources such as **Kaggle**.

## Why Dataset Files Are Not Included?

The original dataset files are not uploaded to this repository because they are very large in size and exceed GitHub's recommended file size limits.

To keep this repository lightweight and accessible, only the project code, notebooks, and documentation are included.
